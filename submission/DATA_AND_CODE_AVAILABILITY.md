# Data and Code Availability Statement

The code, audit logs, release artifacts, and manuscript-supporting files for Boundary Excess Calculus v2.0 are publicly available.

GitHub repository: https://github.com/htetkokokonaing-dev/BECv2

Zenodo DOI: https://doi.org/10.5281/zenodo.21127911

The repository contains Lean 4 / Mathlib source files, GitHub Actions configuration, verification scripts, checksum records, release archives, and manuscript artifacts. The current public CI status is passing.

No private or restricted dataset is required to reproduce the included formal-verification checks.
