# Reproducibility Statement

The public repository provides the reproducibility record for Boundary Excess Calculus v2.0.

Repository: https://github.com/htetkokokonaing-dev/BECv2  
DOI: https://doi.org/10.5281/zenodo.21127911

## Reproducibility components

- Lean 4 / Mathlib source files
- lakefile.toml
- lean-toolchain
- GitHub Actions workflow
- audit logs
- SHA256 checksum records
- release archives
- final monograph artifacts

## Continuous integration

The repository includes a GitHub Actions Lean workflow. The current public workflow status is passing.

## Claim boundary

Reproducibility applies to the included source package, scripts, logs, checksums, and release artifacts. It does not imply that every mathematical statement in the monograph has been independently formalized in Lean.
