## Recommended submission order

1. Use the GitHub repository and Zenodo DOI as public verification records.
2. Use the final PDF as the main manuscript.
3. Use the final DOCX only if the target venue requires DOCX.
4. Include the claim-boundary and reproducibility statements.
5. Submit to the selected venue only after venue-specific formatting is complete.
